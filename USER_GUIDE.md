# 用户指南 - 桌面图片管理工具

## 🎯 应用简介

这是一个简单易用的桌面图片管理工具，主要功能包括：
- 📁 监控文件夹并统计图片数量
- 🔄 按图片数量自动排序
- 🖼️ 图片预览和懒加载
- 💾 配置自动保存
- 🎨 现代化界面设计

## 🚀 快速开始

### 启动应用

打开终端或命令提示符，运行：

```bash
cd D:\Temp\ai-workspace\cc-project\custom-window
npm run start
```

应用会自动构建并启动，显示主界面。

## 📖 详细使用说明

### 第一步：添加文件夹路径

1. 应用启动后，点击顶部工具栏的"添加路径"按钮
2. 在弹出的文件夹选择对话框中，选择一个包含图片的文件夹
3. 点击"选择文件夹"确认
4. 系统会自动扫描该文件夹下的所有子文件夹
5. 统计每个子文件夹中的图片数量

> 💡 **提示**: 可以选择包含多个子文件夹的父文件夹，系统会自动处理所有子文件夹。

### 第二步：浏览文件夹

添加路径后，左侧会显示文件夹列表：

- **排序方式**: 按图片数量从多到少排序
- **显示信息**: 文件夹名称、图片数量、最后修改时间
- **选择文件夹**: 点击任意文件夹名称

### 第三步：预览图片

选择文件夹后，右侧会显示：

1. **大图预览**: 显示第一张图片的完整预览
2. **图片导航**: 使用左右箭头按钮切换图片
3. **图片计数**: 显示当前图片位置（如：3/15）
4. **全屏查看**: 点击图片可以全屏查看

### 第四步：使用图片网格

底部显示图片缩略图网格：

- **懒加载**: 每次显示5张图片，提高性能
- **加载更多**: 点击"加载更多"按钮加载下一批图片
- **进度显示**: 显示加载进度（如：显示 5/50 张，还有 45 张）
- **快速选择**: 点击任意缩略图快速切换到该图片

### 第五步：管理监控路径

1. 点击"管理路径"按钮打开管理界面
2. 查看所有已保存的监控路径
3. 每个路径显示：
   - 文件夹名称和完整路径
   - 子文件夹数量
   - 图片总数
4. 点击删除按钮移除不需要的路径
5. 点击"关闭"保存更改

> 💡 **提示**: 所有路径配置会自动保存，下次启动时自动加载。

### 第六步：刷新数据

- 点击"刷新"按钮重新扫描所有监控路径
- 更新文件夹统计信息
- 获取最新的图片数量

## 🔧 常见问题

### Q: 应用启动很慢？
A: 首次启动需要构建应用，后续启动会很快。如果一直很慢，请检查是否有足够的磁盘空间。

### Q: 图片无法显示？
A: 确保图片格式受支持（jpg, jpeg, png, gif, bmp, webp, svg）。检查文件是否损坏。

### Q: 如何删除所有配置？
A: 删除用户目录下的 `.image-manager-config.json` 文件，然后重启应用。

### Q: 支持哪些操作系统？
A: 支持 Windows、macOS 和 Linux 系统。

### Q: 可以监控网络文件夹吗？
A: 目前只支持本地文件夹。网络文件夹可能需要先映射为本地驱动器。

## 🎯 使用技巧

### 高效工作流

1. **批量添加**: 一次性添加多个相关文件夹路径
2. **定期刷新**: 定期点击刷新按钮更新统计信息
3. **快速浏览**: 使用缩略图网格快速定位目标图片
4. **全屏查看**: 点击图片全屏查看细节

### 性能优化

- 选择包含适量图片的文件夹（建议每个文件夹少于1000张图片）
- 避免选择过大的父文件夹
- 定期清理不需要的监控路径

### 文件组织建议

- 按主题或项目组织图片文件夹
- 使用有意义的文件夹名称
- 定期整理和分类图片
