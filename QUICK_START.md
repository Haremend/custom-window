# 快速启动指南

## 🎯 一句话说明

这是一个桌面图片管理工具，可以按文件夹中图片数量自动排序，支持图片预览和懒加载。

## 🚀 启动步骤

### 方法一：一键启动（推荐）

```bash
npm run start
```

### 方法二：分步执行

```bash

# 1. 构建应用
npm run build

# 2. 启动应用
npm run electron
```

## 📝 使用说明

### 第一步：启动应用

应用启动后会显示主界面，包含：
- 顶部工具栏（添加路径、管理路径、刷新）
- 左侧文件夹列表
- 右侧图片预览区域
- 底部图片网格

### 第二步：添加监控路径

1. 点击"添加路径"按钮
2. 在弹出的对话框中选择包含图片的文件夹
3. 系统会自动扫描该文件夹下的所有子文件夹
4. 统计每个子文件夹中的图片数量

### 第三步：浏览文件夹

- 左侧列表显示按图片数量降序排列的文件夹
- 每个文件夹显示：
  - 文件夹名称
  - 图片数量
  - 最后修改时间
- 点击文件夹名称加载其中的图片

### 第四步：预览图片

- 选择文件夹后，右侧显示第一张图片的大图预览
- 使用左右箭头按钮在图片间切换
- 图片计数器显示当前图片位置（如：3/15）
- 底部显示图片缩略图网格

### 第五步：使用懒加载

- 图片采用懒加载模式，每次显示5张
- 点击"加载更多"按钮加载下一批图片
- 显示进度："显示 5/50 张（还有 45 张）"

### 第六步：管理路径

- 点击"管理路径"按钮打开管理界面
- 查看所有已保存的监控路径
- 可以移除不需要的路径
- 所有配置自动保存，下次启动时自动加载

## 🔧 常见问题

### Q: 应用启动后界面空白？
A: 确保已经添加了至少一个监控路径。点击"添加路径"按钮选择文件夹。

### Q: 图片无法显示？
A: 检查文件格式是否支持（jpg, png, gif, bmp, webp, svg）。

### Q: 如何保存配置？
A: 配置会自动保存，无需手动操作。下次启动时会自动加载之前的路径。

### Q: 如何删除不需要的路径？
A: 点击"管理路径"按钮，在弹出的对话框中可以移除不需要的路径。

## ⚙️ 配置信息

- **配置文件位置**: `~/.image-manager-config.json`
- **支持格式**: jpg, jpeg, png, gif, bmp, webp, svg
- **懒加载数量**: 每页5张图片
- **界面主题**: 现代化蓝白配色

## 🛠️ 开发者信息

### 开发模式

```bash
# 启动开发服务器
npm run dev

# 在另一个终端启动Electron
npm run electron
```

### 构建分发版本

```bash
# 安装 electron-builder
npm install --save-dev electron-builder

# 构建Windows版本
npm run build
npx electron-builder --win
```

## 📞 获取帮助

如果遇到问题，请检查：
1. Node.js版本是否为18+
2. 是否有足够的磁盘空间
3. 选择的文件夹是否包含支持的图片格式
4. 查看Electron控制台是否有错误信息
