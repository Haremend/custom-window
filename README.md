# 桌面图片管理工具

一个基于 Electron + Vue3 + TypeScript 的桌面图片管理工具，支持按文件夹中图片数量进行排序和管理。

## 功能特性

✅ **文件夹监控**
- 添加/删除监控路径
- 自动统计子文件夹中的图片数量
- 支持多种图片格式（jpg, png, gif, bmp, webp, svg）

✅ **智能排序**
- 按图片数量降序排列
- 支持按名称、修改时间排序
- 实时更新统计信息

✅ **图片预览**
- 单图预览模式
- 两图滑动对比模式
- 支持缩放、全屏显示
- 图片导航（上一张/下一张）

✅ **元数据提取**
- 解析图片EXIF信息
- 提取AI生成图片的prompt信息
- 显示图片尺寸、格式、创建时间等
- 支持复制元数据到剪贴板

✅ **现代化界面**
- 基于Element Plus的桌面级UI
- 响应式布局
- 直观的数据可视化

## 技术栈

- **桌面端**: Electron
- **前端框架**: Vue3 + Vite + TypeScript
- **状态管理**: Pinia
- **UI组件**: Element Plus
- **图片预览**: 原生实现 + 全屏支持
- **元数据解析**: exiftool-vendored

## 快速开始

### 环境要求

- Node.js 18+
- npm 或 yarn

### 安装依赖

```bash
npm install
```

### 启动应用

```bash
# 一键启动（构建 + 运行）
npm run start

# 或者分步执行：
# 1. 构建应用
npm run build

# 2. 启动应用
npm run electron
```

### 开发模式（仅开发者需要）

```bash
# 启动开发服务器（用于开发调试）
npm run dev

# 在另一个终端启动Electron进行开发
npm run electron
```

## 使用说明

### 1. 启动应用

```bash
npm run start
```

### 2. 添加监控路径

1. 点击顶部的"添加路径"按钮
2. 选择要监控的文件夹
3. 系统会自动统计该文件夹下所有子文件夹的图片数量

### 3. 浏览文件夹

- 左侧列表显示所有子文件夹，按图片数量降序排列
- 显示每个文件夹的图片数量和最后修改时间
- 点击文件夹名称可以查看其中的图片

### 4. 图片预览

- 选择文件夹后，右侧会自动显示第一张图片预览
- 使用左右箭头按钮在图片间导航
- 点击图片可以全屏查看
- 底部显示图片缩略图网格（支持懒加载）

### 5. 管理路径

- 点击"管理路径"按钮查看所有已保存的路径
- 可以移除不需要的监控路径
- 所有路径配置会自动保存，下次启动时自动加载

### 6. 刷新数据

- 点击"刷新"按钮重新扫描所有监控路径
- 更新文件夹统计信息

## 开发说明

### 主要组件

- **Toolbar**: 顶部工具栏，包含添加路径、刷新等功能
- **FileList**: 左侧文件列表，显示排序后的文件夹
- **ImagePreview**: 图片预览组件，支持单图和对比模式
- **Sidebar**: 右侧信息栏，显示图片元数据

### 状态管理

- **pathStore**: 管理监控路径、文件夹统计、图片列表
- **imageStore**: 管理当前预览图片、对比图片、元数据缓存

### API接口

- **FileSystemAPI**: 封装Electron IPC调用，处理文件系统操作
- **MetadataParser**: 解析图片元数据，支持多种格式
